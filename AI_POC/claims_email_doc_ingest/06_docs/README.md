# Claims Email Doc Ingest System

This repository contains a modular, production-ready solution for insurance claims email ingestion, entity extraction, document analysis, triage, monitoring, and alerting. The system is designed for Databricks, Azure, and SQL Server integration.

## Key Features
- Email ingestion from Azure Blob Storage
- Entity extraction using regex and GenAI
- Document type classification
- Triage and high-risk detection
- SQL Server integration
- Monitoring and alerting (Teams)
- Modular pipelines and unit tests
